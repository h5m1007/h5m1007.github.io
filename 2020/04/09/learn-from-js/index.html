防抖/节流

  • 防抖

当任务频繁触发时，只有任务触发时间间隔超过指定间隔时，任务才会执行 ( 把触发频繁的事件合并一次执行 )

场景：鼠标连续点击，输入框输入联想搜索请求接口

  • 节流

指定时间阈值内，无论触发几次都只会执行一次 ( 保证每 x 毫秒恒定的执行次数 )

场景：滚动监听、鼠标跟随、动画
